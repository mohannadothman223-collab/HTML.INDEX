# Embed Video Exercise

This folder contains the Chapter 5 mandatory exercise solution for "Embed Video".

- `index.html`: embeds a locally hosted video with `loop` and a poster image.
- `styles.css`: centers the video horizontally and vertically on the page.
- `poster.svg`: placeholder poster image for the video.

Currently, the video source is referenced as `WHEATHER.mp4`.

For the solution to work, place your video file in this folder and name it `WHEATHER.mp4`. Then open `index.html` in a browser to see the embedded, looping video.
