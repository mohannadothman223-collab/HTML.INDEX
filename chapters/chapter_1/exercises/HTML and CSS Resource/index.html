HTML & CSS Quick Reference Guide

This page shows common HTML tags and CSS properties with examples for easy review.

Common HTML Tags

Use these tags to structure content and show code examples.

Heading tags

The <h1> to <h6> tags define headings.

<h1>Main title</h1>

Paragraph and text

The <p> tag defines a paragraph.

<p>This is a paragraph.</p>

Lists

Use <ul> for unordered lists and <ol> for ordered lists.

<ul>
  <li>Item one</li>
</ul>

Link

The <a> tag creates a hyperlink.

<a href="https://example.com">Visit site</a>

Image

The <img> tag adds an image.

<img src="image.jpg" alt="Example image">

Code and preformatted text

Use <code> and <pre> for code examples.

<pre><code>Example code</code></pre>

CSS Property Examples

Here are some CSS properties used to style text, layout, and boxes.

Example styles

body {
  background-color: #f0f0f5;
  color: #1a1a1a;
  font-family: Arial, sans-serif;
}

h1 {
  color: #2a4d69;
}

p {
  line-height: 1.6;
}

.container {
  margin: 20px;
  padding: 15px;
  border: 1px solid #ccc;
  background-color: #ffffff;
}

Use a class selector to style a container block.
